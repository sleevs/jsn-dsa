# Algorithms Research and Demonstration

This repository contains implementations and studies of classical computer science algorithms focused on:

- Graph traversal
- Shortest path
- Minimum spanning tree
- String matching
- Network flow
- Greedy strategies

The goal of this project is to explore algorithm behavior, complexity, and practical implementation using Java.

---

## Implemented Algorithms

### Depth-First Search (DFS)
Graph traversal algorithm that explores as far as possible along each branch before backtracking.

Applications:
- Graph traversal
- Cycle detection
- Topological sorting

Developed by Charles Pierre Trémaux.

---

### Breadth-First Search (BFS)
Traverses graph nodes level by level using a queue structure.

Applications:
- Shortest path in unweighted graphs
- Web crawling
- Network broadcasting

Developed by Konrad Zuse (1945).

---

### Dijkstra Algorithm
Finds the shortest path between a source node and all other nodes in a weighted graph with non-negative weights.

Applications:
- GPS navigation
- Routing systems
- Network optimization

Developed by Edsger W. Dijkstra (1956).

---

### Bellman-Ford Algorithm
Shortest path algorithm capable of handling negative edge weights.

Applications:
- Routing protocols
- Financial arbitrage detection

Developed by Alfonso Shimbel, Lester Ford Jr., and Richard Bellman.

---

### Boyer-Moore Algorithm
Efficient string-search algorithm that skips sections of text using heuristic tables.

Applications:
- Text editors
- Search engines
- Pattern matching

Developed by Robert S. Boyer and J Strother Moore (1977).

---

### Floyd-Warshall Algorithm
Computes shortest paths between all pairs of vertices in a graph.

Applications:
- Network routing
- Graph analysis

Developed by Robert Floyd and Stephen Warshall (1962).

---

### Kruskal Algorithm
Finds the minimum spanning tree of a graph using greedy edge selection.

Applications:
- Network design
- Infrastructure optimization

Developed by Joseph Kruskal (1956).

---

### Prim Algorithm
Constructs a minimum spanning tree by expanding from a starting node.

Applications:
- Telecommunications
- Electrical networks

Developed by Vojtěch Jarník (1930).

---

### Maximum Flow Algorithm
Calculates the maximum possible flow in a network graph.

Applications:
- Traffic systems
- Supply chain optimization
- Network capacity analysis

Developed by Ford and Fulkerson (1955).

---

### Topological Sort
Orders vertices in a Directed Acyclic Graph (DAG) according to dependency relationships.

Applications:
- Task scheduling
- Dependency management
- Build systems
